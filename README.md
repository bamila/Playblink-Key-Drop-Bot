# Playblink-Key-Drop-Bot
Python based application which allows to drop steam keys to discord chat with profile, experience, different tier drops and cooldowns. 

=About Project=

This bot was created mainly by enthusiasts who were part of playblink.com website before it was shut down on June 2016. The aim of the bot is to provide similar level or even more advanced experience of "chatbox key drop" function original website had, which provided users with opportunity to drop keys straight into chatbox with the sense of thrill and accomplishment while getting points and reputation  from claimants. However this bot does not give any kind of points as well as it doesn't have mini game implemented in it.
